# Logo File Required

Please add the Krystalore logo as `/public/logo.png`

The logo should be:
- PNG format
- Recommended size: 120x40 pixels for header display
- Transparent background preferred
- Source: https://krystalore.com/wp-content/uploads/2025/10/63d23c1829f84e249e2d8003.png

The header component expects the logo at `/public/logo.png` and will display it at 120x40 pixels in the navigation bar.
